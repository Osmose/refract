Refract Turn any website into an installable application

Enter the full URL to a webpage and click Install to install it as an Open Web App or a Chrome App.

  • Open Web App (Firefox)
  • Chrome App

To finish installing the Chrome app:

  1. Click the "Keep" button when Chrome warns you about the download.
  2. Open the Extensions section in the preferences page.
  3. Ensure that the "Developer Mode" checkbox is checked.
  4. Drag and drop the downloaded crx file over the extensions page. Note that dragging from the downloads bar within Chrome won't work; you must drag from outside of Chrome.

Customization

Refract will attempt to auto-fill any blank customization fields by pulling info from the webpage itself.

512x512 PNG preferred. Other sizes will be scaled to 512x512 pixels.

If you made your own icon, throw it up on Imgur or something and enter the direct link to the image here.
